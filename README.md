# RestAssured-maven-demo

一个使用 maven 工具的 RestAssured 演示项目

测试框架：TestNG
测试报告：allure

**中文** | [English](/README_EN.md)


## 环境准备
- JDK 1.8+ ，我使用的 JDK 19
- maven 3.0+ 我使用的 maven 3.9.5
- RestAssured 4.3.3+，我使用的是最新的 5.3.2 版本

## 技术栈
- RestAssured
- TestNG
- Maven 
- allure
- Github action

## 项目运行

```bash
mvn test
```

## 添加API测试用例

- 在 src/test/java目录下新加测试用例即可
